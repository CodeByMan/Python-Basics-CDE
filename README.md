- `Basics.ipynb` - Main assignment notebook

## Tools Used

- Python
- Jupyter Notebook
- Anaconda

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/CodeByMan/Python-Basics-CDE.git
```

2. Open the project folder:

```bash
cd python-basics
```

3. Start Jupyter Notebook:

```bash
jupyter notebook
```

4. Open `Basics.ipynb`.
